¢ > a > b > c           --------- a b c açılması lazım
```bash
$ echo "> a > b > c" | ./minismet

minismet ¢ > a > b > c
Command:
  outfile: c (append=0)
----
minismet ¢ %     
```


¢ a > b > c             --------- command not found ama b ve c açılacak
```bash
$ echo "a > b > c" | ./minismet

minismet ¢ a > b > c
Command:
  arg[0]: a
  outfile: c (append=0)
----
[!] a : Command not found
minismet ¢ %                                     
```
